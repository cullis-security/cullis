ADR-029 tool-level PDP

Each rule below scopes a single tool name. The POST /v1/policy/tool-call endpoint checks the requesting principal, the model that asked for the tool, and the MCP server the tool lives on against the lists below before allowing the dispatch. Empty list means "no restriction on this axis". A principal in denied_principals beats the allow list. When tool_rules is non-empty the gate becomes explicit-allow: tools that are not listed are denied.

Requires MCP_PROXY_TOOL_PDP_ENABLED=true on the Mastio and CULLIS_CONNECTOR_TOOL_PDP_ENABLED=true on the Connector. Both default to false.
