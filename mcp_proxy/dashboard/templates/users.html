Credentials themselves live at the Source: the Frontdesk container holds multi-user passwords, the Connector desktop holds laptop keypairs, SSO rows are governed at the IdP, and ADR-027 API tokens (culk_*) authenticate OpenAI-compat clients (LibreChat / Cursor / Cherry Studio) without a Source at all — mint them from the user detail page. Create / reset / delete from this page forwards the call to the right canale — the Mastio never sees the plaintext password.
